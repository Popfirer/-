# 世界时钟 · World Clock 2026

基于 Python Tkinter 的多城市世界时钟桌面应用，支持实时天气、节假日查询、2026 年全年日历。

## 功能特性

- **多城市时钟** — 默认显示无锡、首尔、潘普罗纳，支持动态添加/删除/替换城市
- **夏令时自动适配** — 内置欧洲、美国、悉尼三大 DST 规则，自动计算时差
- **实时天气** — 通过 [wttr.in](https://wttr.in) 获取各城市天气（温度、天气描述、天气图标），30 分钟自动刷新
- **节假日提醒** — 通过 [Nager.Date](https://date.nager.at) 获取各国公共假期，当日自动高亮显示
- **2026 年全年日历** — 12 个月网格布局，今日高亮，显示周数，滚动浏览
- **无锡本地信息** — 主界面和日历页顶部实时显示无锡当天日期、星期和周数
- **右键菜单** — 右键点击城市卡片可修改、删除、添加城市，一键恢复默认

## 环境要求

- Python 3.12+
- 操作系统：Windows / macOS / Linux

## 快速开始

```bash
# 1. 创建虚拟环境
python -m venv .venv

# 2. 激活虚拟环境
# Windows:
.venv\Scripts\Activate.ps1
# macOS / Linux:
source .venv/bin/activate

# 3. 运行
python world_clock_v0.1.py
```

> 应用启动后默认置顶显示，窗口大小 1000×780，可自由调整。

## 操作说明

| 操作 | 方式 |
|------|------|
| 查看各城市时间 | 切换到「世界时钟」标签页 |
| 修改城市 | 右键点击城市卡片 → 修改位置，或直接点击城市名 |
| 添加城市 | 右键 → 添加城市 → 选择目标城市 |
| 删除城市 | 右键 → 删除（至少保留一个） |
| 恢复默认 | 右键 → 恢复默认 |
| 浏览日历 | 切换到「2026年日历」标签页，鼠标滚轮滚动 |

## 技术要点

- **GUI 框架**：Tkinter + ttk
- **时区处理**：纯 Python 实现夏令时计算（欧洲/美国/悉尼规则），零外部依赖
- **天气接口**：wttr.in JSON API，线程异步拉取 + 缓存（TTL 30 分钟）
- **节假日接口**：Nager.Date Public Holidays API，缓存（TTL 24 小时）
- **数据刷新**：`after()` 定时器驱动 UI 更新（200ms 间隔），天气/假期后台线程异步刷新
